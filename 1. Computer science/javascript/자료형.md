# 기본 자료형과 객체 자료형의 차이

- 둘다 속성, 메소드를 사용할 수 있음
- 기본자료형의 속성 또는 메소드를 사용할 때 기본 자료형이 자동으로 객체로 변 환됨
- 기본 자료형은 객체가 아니므로 속성과 메소드를 추가할 수 없다
