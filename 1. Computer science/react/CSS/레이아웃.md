# display
display :none 
-> element를 화면에서 숨기기위해 사용

dlsplay: block
-> 블록 단위로 element 를 배치
 <p> <div>,<h1>~<h6> 까지 display 의 기본값이 display:block

display:inline
 element를 라인안에 넣는것
 <span>태그의 display 속성 기본값이 display:inline

display:flex
element를 블록레벨의 flex container로 표시
container이기 떄문에 내부에 다른 element들을 포함
