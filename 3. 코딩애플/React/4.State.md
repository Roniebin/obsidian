# state 를 왜써야함?

#### 변수와 state의 차이점
- 변수 : 유저가 갑자기 수정했을때 자동변경이안됨
- state: 변경시 자동 재랜더링이됨
- 변동시 자동으로 html에 반영되게 만들고싶을때 씀


*** 로고같은것들은 거의안바뀜 -> state안해도됨
