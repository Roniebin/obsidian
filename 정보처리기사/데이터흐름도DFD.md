## **1. 데이터 흐름도 - DFD(Data Flow Diagram)의 개념**

- 데이터 흐름도(DFD)는 시스템 구성요소인 프로세스와 프로세스 간 데이터 흐름을 표현하는 주요 도구
- 자료 흐름 그래프 또는 버블(bubble) 차트라고 부르기도 함
- 구조적 분석 기법에 사용
- 자료 흐름과 기능을 자세히 표현하기 위해 단계적으로 세분화
